🚀 Highly Available Infrastructure for Online Learning Platform
📖 Project Overview

This project demonstrates the deployment of a highly available web infrastructure on AWS for an online learning platform. The architecture is designed to provide redundancy, centralized file sharing, and balanced traffic distribution across multiple web servers.

The solution uses:

* Amazon VPC for network isolation
* Multi-AZ deployment for high availability
* Amazon EC2 instances as web servers
* Application Load Balancer (ALB) for traffic distribution
* Amazon EFS for shared storage
* Security Groups for controlled access
